# Nightly Evolution

每夜自动执行的 workspace 维护 + 社区调研 + memory 蒸馏。

## 设计

- Phase 4: 产出（git push + Discord 摘要）

- 编辑 symlink 文件前必须 readlink -f
- Discord 报告必须用 Components V2 卡片
